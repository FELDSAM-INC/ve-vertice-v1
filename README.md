Gulpd
=================

Gulpd is a node daemon for megam cloud. Builders supported are sbt, mvn, npm, bundler). 

* `sbt `
* `mvn`
* `bundler`
* `npm`


### Requirements

> 
[Golang 1.2o > +](http://www.golang.org)

#### Tested on Ubuntu 13.04, AWS - EC2, GCE



## Usage

gulpd start 
 
gulpd start
